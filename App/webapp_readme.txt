---== HOW TO USE WEBSERVICE ==---

1. Install Tomcat: https://tomcat.apache.org/

2. Move game_service.war to ~/apache-tomcat-x.xx.xx/webapps

3. Launch ~/apache-tomcat-x.xx.xx/bin/startup.sh

4. The service should now be running at http://localhost:8080/game_service/

5. Use ifconfig to find out your ip address for multiplayer games

6. Have fun ;)
